4. Write a short note on Tag and element with an examples.
Ans: 
HTML Tags: Tags are starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags.
Example- <b>  </b>
HTML Elements: Elements enclose the contents in between the tags. They consists of some kind of structure or expression. It generally consist of a start tag, content and an end tag.
Example: <b> This is the content </b>
